# 🚀 NoWait - Queue Management System

## 📌 Overview

NoWait is a real-time queue management system designed to reduce waiting time in hospitals, banks, and service centers.

Users can generate tokens and track live queue updates, while admins can manage and control the queue efficiently.

---

## ⚙️ Features

### 👤 User

* Generate token
* View live queue status
* Track current serving number

### 🖥️ Admin

* Call next token
* Manage queue in real-time

### ⚡ Real-Time System

* Instant updates using Socket.io
* Sync across multiple users

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Socket.io Client

### Backend

* Node.js
* Express.js
* Socket.io
* MongoDB (planned)

---

## 📂 Project Structure

```
frontend/   → React App
backend/    → Node Server
```

---

## 🚀 Getting Started

### 1. Clone Repository

```
git clone https://github.com/your-username/nowait-queue-management-system.git
```

### 2. Install Frontend

```
cd frontend
npm install
npm run dev
```

### 3. Run Backend

```
cd backend
node server.js
```
